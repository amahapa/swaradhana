Pitch Behavior

When you change the global Key, the tabla's Dayan (treble drum) automatically re-tunes to match — in the middle octave only. The Bayan (bass drum) stays at a constant low frequency. Changing the global Saptak (octave) does not affect the tabla.

For real tabla samples, the app pitch-shifts the audio to match your selected key. Samples recorded in Key of E are shifted proportionally (e.g., selecting G shifts up by ~3 semitones).

Swara Notation (Notes)

Hindustani music uses 7 natural notes (shuddha swaras) and 5 altered notes:

Symbol Name Type
SSa (Shadaj)Fixed
RRe (Rishabh)Shuddha
rKomal ReFlat
GGa (Gandhar)Shuddha
gKomal GaFlat
mMa (Madhyam)Shuddha
MTivra MaSharp
PPa (Pancham)Fixed
DDha (Dhaivat)Shuddha
dKomal DhaFlat
NNi (Nishad)Shuddha
nKomal NiFlat

Octave markers:

  • P, D, N, — Lower octave (Mandra) — comma suffix
  • S R G m P D N — Middle octave (Madhya) — no suffix
  • S' R' G' m' P' — Upper octave (Taar) — apostrophe suffix

Flute positions 1-15: P,(1) D,(2) N,(3) S(4) R(5) G(6) m(7) P(8) D(9) N(10) S'(11) R'(12) G'(13) m'(14) P'(15)

Thaat (Parent Scales)

A Thaat defines which notes are natural, flat, or sharp. There are 10 thaats. Changing the thaat on the main screen changes the notes used in exercises.

Creating Exercises (Alankaar)

An Alankaar is a melodic pattern practiced ascending (Aroha) and descending (Avaroha) across the scale.

How to create:

  1. Tap the Exercise bar on the main screen
  2. Tap + New Exercise
  3. Select a Taal, Start/End notes, and Level
  4. Type your pattern and tap Generate
  5. Review the Aroha/Avaroha tables and tap Save

Pattern input format:

Use digits 1-9 for relative note positions. Spaces separate vibhags to match the taal structure.

Symbol Meaning Example
1-9Note positions1234 = 4 notes ascending
.Rest (silence)12.4 = play, play, rest, play
_Sustain previous note12_3 = note 2 held for 2 beats
spaceVibhag separator1234 2345 = 2 vibhags
[...]Multiple notes per beat[12] = 2 notes in 1 beat (dugun)

Examples:

1234 2345 3456 4567
Teentaal, sequential ascending, ekgun
123 [22][33] [33][44]
Rupak, ekgun + dugun mixed
12_3 23_4 34_5 45_6
Teentaal, with sustained notes

The app generates the full Aroha (ascending) and Avaroha (descending mirror) automatically.

Taal (Rhythm Cycles)

A Taal is a repeating rhythm cycle of beats (matras) grouped into sections (vibhags).

Key markers in the taal table:

  • X (Sam) — The heaviest beat. Start of the cycle. Amber highlight.
  • 2, 3, 4 (Tali) — Clap beats. Medium weight. Teal highlight.
  • 0 (Khali) — Wave/open beats. Softest. Purple highlight.

Taal Variations

You can create custom tabla bol patterns while keeping the taal's structure (beats, vibhags, Sam/Khali positions).

How to create a variation:

  1. Open Taal List (tap the Taal tile)
  2. Tap the > chevron next to any taal
  3. Tap Copy on the Default theka, or + New Variation
  4. Tap any matra cell to select it
  5. Tap bols from the palette to build that matra (up to 4 bols per beat)
  6. Tap Save

Switch between variations using the < D > arrows on the Taal tile.

Each matra can have 1-4 bols. Multiple bols in one matra split the beat time equally.

Laykari (Speed/Subdivision)

Laykari controls how many notes are played per beat:

  • Ekgun (1x) — 1 note per beat: S R G m
  • Dugun (2x) — 2 notes per beat: [SR] [Gm] [PD] [NS']
  • Tingun (3x) — 3 notes per beat: [SRG] [mPD] [NS'R']
  • Chaugun (4x) — 4 notes per beat: [SRGm] [PDNS']

In the pattern input, use brackets to indicate multiple notes per beat. You can mix laykari within the same cycle — for example, ekgun in one vibhag and dugun in another.

Exercise Modes

When an exercise is loaded, choose a mode from the main screen:

  • Demo — App plays tabla + tanpura + swaras automatically. Listen and learn the pattern.
  • Practice — App plays one phrase with swaras, then goes silent for one cycle. You play/sing the same phrase. Table highlights continue as visual cues. Tabla and tanpura never stop.
  • Test — Coming soon. Your playing will be recorded and evaluated.

Tip: Tap the exercise name on the main screen to see the live taal table with note highlighting while the exercise plays.

Key, Tanpura & Instruments

Key: Sets the base Sa frequency for your flute. Tap the Key display in the top bar to change. Default is E (164.81 Hz).

Tanpura: The drone that provides the tonal reference. Tap the Tanpura tile to configure string tuning (Pa/Ma/Ni), octave, speed, and advanced settings (jivari, variance, reverb).

Tabla: Choose between electronic synthesis or real tabla samples in Settings > Customize Tabla. Adjust bass/treble EQ in the Volume page.

Swara Instruments: Go to Settings > Customize Swara to select instruments (Harmonium, Strings, Acoustic Guitar, Piano). You can select multiple for layered sound. Each has its own volume slider.

Backup & Restore

Your settings, exercises, and taal variations are saved in your browser automatically. To back up or transfer your data:

  1. Go to Settings (gear icon in top bar)
  2. Export Data — Downloads a JSON backup file
  3. Import Data — Restores from a previously exported file
  4. Clear All Data — Resets everything to defaults (cannot be undone)

Data is stored in your browser's local storage. It persists across sessions but is lost if you clear browser data.
